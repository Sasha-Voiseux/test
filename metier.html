1)Métier choisi

Comme métier j'ai choisi développeur web.


2)En quoi consiste ce métier

Le développeur web effectue la réalisation technique et le développement informatique d'un site web. A l'aide du cahier des charges réalisé par le chef de projet, le développeur web programme les fonctionnalités qui correspondent aux besoins du client pour son site web.
Le développeur web peut exercer dans une ESN (Entreprise de Services du Numérique), dans une agence Web, en tant que travailleur indépendant ou directement chez le client. Certains développeurs web sont également bénévoles dans le cadre de projets liés à l'informatique libre.


3)En quoi est-il utile à la société

Sans le développeur web, les entreprises, associations, etc... ne pourraient pas se faire connaitre et se développer aussi rapidement.


4)Qualités requises/Exigences

Les qualités requises pour devenir développeur web sont:

  • Etre passionné
  • Etre patient
  • Savoir se concentrer
  • Aimer chercher les erreurs (là où on les a faites) et les corriger
  • Aimer créer des choses

Il faut aussi être titulaire d'un bac +2 à bac +5 en programmation web.
